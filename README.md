## 📖 简介

Skill Executor 是一款智能任务执行助手，通过工具调用高效完成各类任务。支持代码生成、文件操作、命令执行、多媒体处理等功能，并可加载自定义技能扩展能力。

### 适用场景

- 💻 **软件开发** - 代码生成、重构、调试
- 📁 **文件管理** - 批量处理、格式转换、内容搜索
- 🎵 **多媒体制作** - 课件生成、语音合成、视频处理
- 📊 **数据分析** - 数据处理、可视化、报告生成
- 🔧 **系统运维** - 自动化脚本、批量操作

## 📥 下载与安装

### Windows

1. 访问发布页面下载 `Skill-Executor-setup-x64.exe`
2. 双击运行安装程序
3. 按照向导完成安装
4. 在开始菜单或桌面找到图标，双击启动

### macOS

1. 访问发布页面下载 `Skill-Executor-x64.dmg`（Intel）或 `Skill-Executor-aarch64.dmg`（Apple Silicon）
2. 双击打开 DMG 文件
3. 将应用图标拖拽到「应用程序」文件夹
4. 在启动台或应用程序文件夹中找到并打开

### Linux

1. 访问发布页面下载对应版本的 `.deb`（Debian/Ubuntu）或 `.rpm`（Fedora/CentOS）或 `.AppImage`
2. 安装并运行：
   ```bash
   # Debian/Ubuntu
   sudo dpkg -i skill-executor_xxx_amd64.deb

   # Fedora/CentOS
   sudo rpm -i skill-executor-xxx.x86_64.rpm

   # AppImage（通用）
   chmod +x Skill-Executor-x86_64.AppImage
   ./Skill-Executor-x86_64.AppImage
   ```

## 🚀 快速开始

### 第一步：配置 AI 模型

首次启动后，需要配置 AI 模型才能使用：

1. 点击左上角 **⚙️ 设置** 按钮
2. 选择 **模型配置** 标签页
3. 选择您的 AI 服务提供商：

   **选项 A：Anthropic Claude（推荐）**
   - 获取 API Key：访问 https://console.anthropic.com
   - 输入 API Key
   - 选择模型（推荐 `claude-sonnet-4-20250514`）

   **选项 B：OpenAI**
   - 获取 API Key：访问 https://platform.openai.com
   - 输入 API Key
   - 选择模型（推荐 `gpt-4o`）

   **选项 C：阿里云百炼（国内用户推荐）**
   - 获取 API Key：访问 https://bailian.console.aliyun.com
   - 输入 API Key
   - 选择模型（推荐 `qwen-max`）

4. 点击 **测试连接** 验证配置
5. 点击 **保存**

### 第二步：开始对话

1. 在底部输入框中输入您的需求，例如：
   - "帮我创建一个 Python 脚本，读取 CSV 文件并生成图表"
   - "把这个函数改成异步的"
   - "查找项目中所有使用 axios 的地方"
   - "生成一个教学课件，讲解重力概念"

2. 按 **Enter** 发送，或点击 **发送** 按钮

3. AI 将自动分析并执行任务，过程中可能：
   - 📝 生成代码
   - 📂 读取/修改文件
   - 💻 执行命令
   - 🎵 合成语音
   - ❓ 向您确认某些操作

### 第三步：查看结果

- 对话区域会实时显示 AI 的思考过程和输出
- 文件修改会自动保存到磁盘
- 命令执行结果会直接显示在对话中
- **需要选择时会弹出选项对话框**

## 🎯 Skill 技能系统

Skill 是预定义的自动化工作流，让 AI 按特定方式执行任务。

### 技能目录

| 目录 | 说明 |
|------|------|
| `system/skills/` | 系统内置技能（只读） |
| `skills/` | 用户自定义技能 |

### 内置技能示例

- **课件生成** - 自动生成教学课件（配音+动画）
- **代码审查** - 检查代码质量、安全性、性能
- **文档生成** - 为代码自动生成文档

### 自定义技能

在 `skills/` 目录下创建 `.md` 文件：

```markdown
# 技能名称
简短描述这个技能的作用

TRIGGER
- 触发关键词1
- 触发关键词2

PROMPT:
详细的提示词内容，定义 AI 如何执行这个技能
```

用户技能可以覆盖同名的系统技能。

## ✨ 功能介绍

### 📁 文件操作

支持读取、创建、修改项目中的任何文件：

- **读取文件**：自动查看文件内容
- **创建文件**：生成新文件并写入内容
- **修改文件**：精确定位并替换代码
- **搜索文件**：快速定位特定内容的文件

### 💻 命令执行

在安全前提下自动执行 Shell 命令：

```
用户：运行 npm install 并启动开发服务器
```

> ⚠️ **危险命令确认**：删除、格式化等危险操作会弹出确认对话框，需您手动批准。

### 🎵 多媒体处理

- **文字转语音 (TTS)**：支持多种中英文音色
- **媒体播放**：直接在对话中播放音频/视频/图片
- **课件生成**：结合 TTS 和动画生成教学视频

### 🔍 内容搜索

快速查找代码库中的内容：

- **全文搜索**：查找包含特定文本的位置
- **文件搜索**：按文件名或扩展名查找

### ✅ 任务管理

复杂任务会自动分解为待办清单，实时跟踪进度：

```
[✓] 分析项目结构
[✓] 创建数据库模型
[ ] 编写 API 接口
[ ] 添加单元测试
```

### 🌐 网络功能

- **Web 搜索**：查询最新文档和技术信息
- **网页抓取**：获取指定网页的内容

## 🔧 高级设置

### 工作目录

默认使用用户主目录，可自定义：

1. 点击 **⚙️ 设置** → **常规**
2. 在 **工作目录** 中选择项目根目录
3. 点击 **保存**

### 模型参数

- **温度（Temperature）**：控制输出随机性（0-1，越高越有创意）
- **最大令牌数**：限制单次响应的长度
- **超时时间**：设置请求超时（毫秒）

### 主题切换

- **浅色模式**：适合明亮环境
- **深色模式**：适合暗光环境，保护视力
- **跟随系统**：自动匹配系统主题

## ❓ 常见问题

### Q: 为什么连接不上 AI 服务？

A: 请检查：
1. API Key 是否正确
2. 网络连接是否正常
3. 是否选择了正确的 API 端点
4. 防火墙是否阻止了连接

### Q: 命令执行失败怎么办？

A:
- 检查命令语法是否正确
- 确认是否有执行权限
- 查看错误信息定位问题
- 某些命令可能需要管理员权限

### Q: 文件修改后能撤销吗？

A: 当前版本不支持自动撤销，建议：
- 使用 Git 等版本控制工具
- 重要修改前先备份
- 在对话中让 AI 帮您回滚

### Q: 如何创建自定义技能？

A: 在 `skills/` 目录下创建 `.md` 文件，按照技能格式编写即可。用户技能会覆盖同名的系统技能。

### Q: 支持离线使用吗？

A: 不支持。本应用需要联网调用 AI 服务，但所有文件操作都在本地完成。

## 🔒 隐私与安全

- **本地优先**：所有文件操作在本地完成，不会上传您的代码
- **API Key 加密存储**：凭据加密保存在系统密钥链中
- **危险操作确认**：删除、格式化等命令需手动批准
- **会话数据本地存储**：对话记录保存在本地，不会同步到云端
