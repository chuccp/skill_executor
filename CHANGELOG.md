## [1.0.0] - 2025-01-XX

### Added
- Initial release
- AI-powered skill execution platform
- Multi-turn conversation support
- Tool calling capabilities (file operations, web search, shell commands)
- TTS (Text-to-Speech) support
- Media playback (audio, video, images)
- Skill system with custom prompts
- Context management with auto-compression
- Token usage tracking

### Technical
- Built with Tauri v2
- Vue 3 frontend with TypeScript
- Express backend with WebSocket support
